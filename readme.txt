In this repository, you can find code example projects for the AURIX™ Development Studio.
Each code example is provided with a tutorial, which can be downloaded from: https://www.infineon.com/aurix-expert-training
To import a code example, use the import function of the AURIX™ Development Studio: File >> Import... >> Infineon >> "AURIX Project from Code Examples", select "GitHub hosted Examples Repository" and select the code example to be imported. 
You can download the AURIX™ Development Studio from here: https://www.infineon.com/aurixdevelopmentstudio
